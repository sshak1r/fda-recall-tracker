About This Project

What It Does

RecallWatch lets you search, filter, and visualize FDA food recall data. You can save recalls to a personal watchlist stored in a database.

Tech Stack

  • Backend: Node.js + Express
  • Database: Supabase (PostgreSQL)
  • External API: openFDA Food Enforcement API
  • Charts: Chart.js
  • Hosting: Vercel

API Endpoints

  • GET /api/recalls — Search FDA recall data
  • GET /api/saved — Get saved recalls from database
  • POST /api/saved — Save a recall to database
  • DELETE /api/saved/:id — Remove a saved recall

Target Browsers

Chrome, Firefox, Safari, and Edge (latest versions). Works on mobile too.
